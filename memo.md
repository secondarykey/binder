# binder is markdown editor

binderの目標は、技術文章をテキストで作成する為のエディタです。

ノートを追加して、マークダウンで文章を書いていき画像は mermaid で書いていきます。
もちろん既存のデータも追加できます。

特徴としてはそれらを自動でGitで管理しているところで、
画像の変更も抑えることができます。

また、構成をGitHub Pagesに合わせている為、それをそのまま公開することができます。
