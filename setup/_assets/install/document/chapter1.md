# Chapter 1

Binderはノートに子データを設定でき、そのページに関連する他のノートやダイアグラムを指定できます。

という風に書くことで、図を表示できます。
".binderSVG"クラスに書いてあるので、border などでデザインすることも可能です。


ダイアグラムにはスタイルテンプレートというものがあり、BaseTheme に表示モードが設定されていて、そちらで表示モードを変更できるようにしていますし、タイアグラムのスタイルテンプレートを変更して、テーマやスタイルを様々変更することが可能です。
